=== かんたん不動産AI査定 ===

匿名の不動産価格査定フォーム。国交省「不動産情報ライブラリ」の実成約事例から
参考価格レンジを算出し、結果をメールで送信＋顧客情報を保存します。

※ 詳しい使い方は、管理画面の「匿名不動産AI査定 → 設定 → 使い方」タブにも
   同じ内容が載っています。


■ 導入手順
1. 「プラグイン」→「新規追加」→「プラグインのアップロード」→ この zip を
   選択してインストール →「有効化」
2. 左メニュー「匿名不動産AI査定 →設定」で以下を設定
   - APIキー（不動産情報ライブラリの Ocp-Apim-Subscription-Key）
     ※未入力ならモックデータで動作確認できます
   - サイト名／運営者名／問い合わせ先／送信元メール
   - プライバシーポリシーURL・利用規約URL（既存固定ページのURL）
3. 査定を出したい固定ページに、ショートコードを貼る

       [fudosan_satei]


■ ショートコード（デザインパターン）

   [fudosan_satei]                       標準。全項目・幅100%・枠なし
   [fudosan_satei design="compact"]      必須＋築年のみ。幅440pxのカード
   [fudosan_satei design="card"]         全項目を枠＋影のカードで表示
   [fudosan_satei design="teaser" url="/satei/"]
                                         3項目だけ入力させ、値を引き継いで
                                         url のページ（本フォーム）へ遷移

■ ステップ構成にする場合（推奨）
   1) トップのメインビジュアル横に
      [fudosan_satei design="teaser" url="/satei/" logo="ロゴ画像URL"
         title="60秒でかんたん入力！" subtitle="査定結果はメールでお届け"
         button="無料査定スタート"]
   2) 査定ページ（/satei/）に
      [fudosan_satei]
   トップで選んだ 物件種別・都道府県・市区町村 が引き継がれます。

■ 属性
   design   : default（省略時） / compact / card / teaser
   url      : teaser のみ必須。ボタンの遷移先URL
   button   : ボタン文言
   title    : teaser の見出し
   subtitle : teaser の小見出し
   logo     : teaser のロゴ画像URL（ロゴ左・見出し右の横並びになる）
   note     : teaser のフォーム下に出す小さな注記（省略時なし）
   fields   : teaser に出す項目をカンマ区切りで指定（省略時 ptype,pref,city）
              使える値: purpose / ptype / pref / city / area / build_year
              書いた順に並ぶ。city を入れると pref が自動で補われる。
              選んだ項目はすべてフル入力フォームに引き継がれる。
              例) fields="purpose,ptype,pref,city"


■ 設定タブ
   基本設定           : APIキー・運営者情報・送信元メール・ポリシーURL
   表示項目・対象エリア : 任意項目の表示ON/OFF、対象都道府県の限定（空なら全国）
   自動返信メール      : 件名・本文を差し込みタグでカスタマイズ／テスト送信ボタン
   使い方             : 本ファイルと同じ内容

■ 顧客情報（査定リード）
   左メニュー「匿名不動産AI査定 → 査定結果・顧客情報」
   CSVエクスポート（Excel向け）と、行ごとの削除（個人情報の削除依頼対応）に対応。
   データは自サイトのDB（wp_fudosan_satei_leads）にのみ保存され、外部送信されません。
   （査定APIには物件条件のみ送信。メールアドレスは送信しません）

■ 査定ロジック
   市区町村の直近8四半期の成約事例を取得し、
   「地区（町名）」→「面積±30%」→「築年±10年」の順に絞り込み、
   ㎡単価の四分位（25%〜75%）で参考価格レンジを算出します。
   条件に近い事例が5件未満の場合は査定を行わず、個別査定へ案内します。
   市区町村を選んだ時点で種別ごとの事例数を表示し、少なければ事前に警告します。

■ メール到達率について
   サーバーから直接送るメールは迷惑メール判定されやすいため、
   「WP Mail SMTP」等で SMTP 送信にし、送信ドメインの
   SPF / DKIM / DMARC を設定してください。
   設定→自動返信メールタブの「テストメールを自分宛に送信」で確認できます。

■ 通信方式
   フォーム送信は admin-ajax.php 経由です（REST APIを無効化している
   セキュリティ設定〔SiteGuard等〕の環境でも動作します）。

■ 法的な注意
   本プラグインが出すのは宅建業の「価格査定（参考価格）」であり、
   不動産鑑定士による「鑑定評価」ではありません。画面・メール・免責文で
   明示しています。免責文は削除しないでください。
   公開前に弁護士等の確認を推奨します。
